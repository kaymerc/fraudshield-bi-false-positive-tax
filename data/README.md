# Data Setup

This project uses the public **Credit Card Transactions Fraud Detection Dataset** hosted on Kaggle and generated with the Sparkov simulation tool.

## Required Files

Download the dataset and place these two files in this directory:

```text
fraudTrain.csv
fraudTest.csv
```

Dataset source: https://www.kaggle.com/datasets/kartik2112/fraud-detection

The files are intentionally excluded from GitHub because the training file is approximately **335 MB** and the testing file is approximately **143 MB**. GitHub's standard per-file limit also prevents committing the training file directly.

The dataset is simulated. This repository contains no employer-owned records, real customer records, internal fraud rules, or confidential business information.
