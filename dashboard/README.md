# Power BI Dashboard

The FraudShield BI report contains four pages:

1. **Executive Overview** — compares logistic regression and gradient boosting at the 0.50 threshold.
2. **Threshold Decision Lab** — shows how precision, recall, F1-score, false positives, and review volume change as the threshold changes.
3. **Cost & Recommendation** — compares low-, moderate-, and high-cost scenarios and identifies the recommended model and threshold.
4. **False-Positive Deep Dive** — displays false-positive rates by transaction category and fraud rates by transaction hour.

## Files

- `FraudShield_BI_Dashboard.pbix` — editable Power BI Desktop report
- `FraudShield_BI_Dashboard.pdf` — four-page view-only export
- `executive_overview.png` — page 1 preview
- `threshold_decision_lab.png` — page 2 preview
- `cost_and_recommendation.png` — page 3 preview

Open the `.pbix` file with Power BI Desktop. The PDF and PNG previews can be viewed directly on GitHub without Power BI.
